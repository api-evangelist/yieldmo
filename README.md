# Yieldmo

Yieldmo is a programmatic native advertising marketplace and smart exchange that differentiates and enhances the value of ad inventory for buyers and sellers. Founded in 2012 and headquartered in New York, the company operates as a supply-side platform (SSP) and ad exchange connecting premium publishers with advertisers across Display, Native, High-Impact, Video, and CTV formats.

The YMax platform — Yieldmo's core technology — uses machine learning and attention analytics to process 100+ TB of data daily, capturing 80+ data signals per impression to drive creative optimization, contextual targeting, and inventory curation. The platform operates cookieless and ID-independent, enabling privacy-compliant audience identification.

## APIs

- **JavaScript SDK API** — Publisher-side SDK enabling ad placement management and virtual page view requests for single-page applications via `ads.yieldmo.com`.
- **Prebid.js Bid Adapter** — Header bidding integration supporting display and video inventory with placement IDs, optional bid floors, and user ID module compatibility.
- **Synthetic Inventory Module** — Prebid.js module for Yieldmo Synthetic Outstream ad injection via Google Ad Manager integration.
- **YMax Platform / Exchange API** — Programmatic exchange for curated inventory, PMP deals, and DSP integration including The Trade Desk Easy Proposal API workflow.
